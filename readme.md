- 概念
[√] 后台管理系统的核心在于权限管理，其他业务都是基于权限来展开
- 方案
权限管理不同角色对应不同权限，用户登陆之后拿到登录凭证，再去请求权限接口，筛选出当前用户的动态路由
菜单权限依赖vue-router的动态路由
按钮权限可以使用vue自定义指令或者v-if实现，权限可以存放在路由meta字段里，也可以分开单独接口请求
- 动态路由刷新丢失
页面刷新动态路由丢失，全局路由钩子拦截获取动态路由，获取不到则通过vuex发送接口重新获取
